## Getting Started

To download our project:

    git clone git@github.com:fac-17/.git
    
    cd 
    
    npm i
    
    touch .env
    
Then add your Yelp API key to your ```.env``` file like this:

    YELP_API_KEY=[YOUR API KEY HERE]

To run the project locally:

    npm start
